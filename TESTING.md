# TreeD Testing Documentation

## Manual Test Cases

### Test Case 1: Add Tree
**Steps:**
1. Open the app
2. Tap "Add Tree" button
3. Fill in tree species: "Oak"
4. Fill in height: "15"
5. Fill in diameter: "2"
6. Tap "Get Current Location"
7. Tap "Add Tree"

**Expected Result:** Tree appears on map with marker

**Status:** ✅ PASS

### Test Case 2: View Tree List
**Steps:**

**Expected Result:** All added trees are displayed

**Status:** ✅ PASS
